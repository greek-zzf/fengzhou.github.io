变量名的力量

发表于 2019-11-21 更新于 2020-04-18 分类于 编码技巧

1.变量名的力量

  • 为变量命名时，该名字要完全，准确地描述出该变量所代表的食物

  • 以问题为导向

  • 最适当的平均长度

    • 平均10~16个字符
    • 平均8~20个字符

  • 变量名对作用域的影响

    • 较长的名字适用于很少用到的变量或者全局变量
    • 较短的名字适用于局部变量或者循环变量

  • 变量名中的计算值界定

    • 类似于：Total，Sum，Average，Max，Min，Record，String，Pointer。把界定词加到名字的最后
    • Num限定词例外

  • 变量名中的常用对仗词

    • begin/end
    • first/last
    • locked/unlocked
    • min/max
    • next/previous
    • old/new
    • opened/closed
    • visible/invisible
    • source/target
    • source/destination
    • up/down

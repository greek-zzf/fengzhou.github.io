--- ### CentOS7安装MySQL(以5.7版本举例，其他版本以此类推)

1.去下载MySQL5.7安装包，直接点击No thanks,just start my download就可以开始下载了。

2.通过rpm -qa | grep mariadb命令查看 mariadb 的安装包

3.通过rpm -e mariadb-libs-5.5.56-2.el7.x86_64 --nodeps命令装卸 mariadb。

4.通过 cd /usr/local/ 命令进入根目录下的usr目录下的local目录，并创建mysql文件夹mkdir mysql

5.把下载好的安装包上传到mysql文件夹,通过tar -xvf mysql-5.7.27-1.el7.x86_64.rpm-bundle.tar解压安装包。

6.解压好后，依次执行如下命令。

rpm -ivh mysql-community-common-5.7.27-1.el7.x86_64.rpm --nodeps --force

rpm -ivh mysql-community-libs-5.7.27-1.el7.x86_64.rpm --nodeps --force

rpm -ivh mysql-community-client-5.7.27-1.el7.x86_64.rpm --nodeps --force

rpm -ivh mysql-community-server-5.7.27-1.el7.x86_64.rpm --nodeps --force

7.依次执行如下命令，执行MySQL的初始化和相关配置。

mysqld --initialize;

chown mysql:mysql /var/lib/mysql -R;

systemctl start mysqld.service;

systemctl enable mysqld;

8.通过 cat /var/log/mysqld.log | grep password命令查看MySQL数据库的密码

9.之后同Windows一样登录MySQL即可,修改密码指令也和Windows一样。

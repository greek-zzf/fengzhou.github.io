​ 2.调用builder的setter方法设置可选参数

简单的解释：首先使用new Student.Builder()方法，构造Builder这个内部类，由于构造方法有id和name两个参数，所以需要填写对应信息。之后，构造出来的Builder对象，由于内部类Builder里面定义了很多给属性赋值的方法，所以用刚才新建的内部类对象调用对应方法，并填上相关信息，例:.age(18)。进入age方法里面后，有个return this语句，又把我们建的对象返回了，其他参数方法以此类推。最后，我们可以调用build()方法，把赋值好的builder对象当做参数，调用Student的构造方法，并把builder对象的属性值赋给Student的属性，并把构造好的Student对象返回给student2,之后运行测试就会有我们刚才设置的信息
